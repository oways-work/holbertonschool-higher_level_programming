The ``1-my_list`` module
========================

Using ``MyList``
----------------
Import the class:
    >>> MyList = __import__('1-my_list').MyList

Instantiate the class:
    >>> my_list = MyList()

Check if it inherits from list:
    >>> isinstance(my_list, list)
    True

Test with integers:
    >>> my_list.append(1)
    >>> my_list.append(4)
    >>> my_list.append(2)
    >>> my_list.append(3)
    >>> my_list.append(5)
    >>> print(my_list)
    [1, 4, 2, 3, 5]

Test the print_sorted method:
    >>> my_list.print_sorted()
    [1, 2, 3, 4, 5]

Verify that the original list is still unsorted:
    >>> print(my_list)
    [1, 4, 2, 3, 5]

Test with negative numbers:
    >>> my_list_neg = MyList([-1, -5, 0, 10, 2])
    >>> my_list_neg.print_sorted()
    [-5, -1, 0, 2, 10]

Test with an empty list:
    >>> empty_list = MyList()
    >>> empty_list.print_sorted()
    []
