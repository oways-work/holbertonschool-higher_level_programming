# SQL - More Queries

## Description
This project focuses on advanced SQL topics, including:
* Managing user privileges (DCL - Data Control Language)
* Table constraints (NOT NULL, UNIQUE, PRIMARY KEY)
* Relationships and Foreign Keys
* Advanced JOIN operations (INNER, LEFT)
* Aggregating data with GROUP BY
