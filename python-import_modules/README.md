# Python - Import & Modules

This project focuses on learning how to import functions and files in Python, how to create modules, and how to use command line arguments using `sys.argv`.

## Requirements
* Allowed editors: `vi`, `vim`, `emacs`
* All files are interpreted/compiled on Ubuntu 20.04 LTS using python3 (version 3.8.5)
* All files end with a new line
* The first line of all files is exactly `#!/usr/bin/python3`
* Code uses the `pycodestyle` (version 2.8.*)
* All files must be executable

## Files

| Filename | Description |
| -------- | ----------- |
| `0-add.py` | Imports the function `add(a, b)` from the file `add_0.py` and prints the result of the addition `1 + 2 = 3`. |
| `1-calculation.py` | Imports functions from `calculator_1.py`, does some basic math operations, and prints the results. |
| `2-args.py` | Prints the number of and the list of its command line arguments. |
| `3-infinite_add.py` | Prints the result of the addition of all command line arguments. |
| `4-hidden_discovery.py` | Prints all names defined by the compiled module `hidden_4.pyc`. |
| `5-variable_load.py` | Imports the variable `a` from the file `variable_load_5.py` and prints its value. |
